# Sales Data Analysis Dashboard

This project is an interactive sales analytics dashboard built using Python and Streamlit.

## Features
- Data visualization of sales trends
- Revenue analysis
- Date filtering
- Downloadable filtered dataset as CSV
- Downloadable Excel report 

## Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib

## How to Run

1. Install dependencies
pip install -r requirements.txt

2. Run the dashboard
streamlit run sales_analytics_dashboard.py

## Project Purpose
This project demonstrates data analysis and visualization skills using Python.
